# Phishing Detection Project

This project loads a phishing dataset, cleans it, trains a phishing detection model, and saves the results.

## Files

- `Phishing.py` - main script that loads data, cleans it, trains a RandomForest model, evaluates it, and saves outputs.
- `phishing_Dataset/dataset_phishing.csv` - original dataset file.
- `cleaned_dataset.csv` - cleaned dataset generated by the script.
- `phishing_model.joblib` - trained model saved by the script.
- `requirements.txt` - Python package dependencies.


## What the script does

1. Loads `phishing_Dataset/dataset_phishing.csv`
2. Removes missing values and duplicate rows
3. Encodes categorical columns
4. Normalizes numeric features
5. Saves cleaned data to `cleaned_dataset.csv`
6. Trains a `RandomForestClassifier`
7. Prints accuracy and classification report
8. Saves the trained model to `phishing_model.joblib`

## Notes

- The script expects the dataset at `phishing_Dataset/dataset_phishing.csv`.
- If you add a new dataset path or dataset file, update `Phishing.py` accordingly.
- You can improve the project by splitting preprocessing and model training into functions, adding command-line arguments, or building a prediction function for new URLs.
